# Employee-Management-Analysis
The Employee Management System is a structured database project that manages employee, department, payroll, leave, and qualification information in an organized manner. It helps analyze workforce distribution, salary trends, leave patterns, and overall payroll expenditure to support informed business decisions.


Procedure of the Project
### Understanding the Dataset
Reviewed the provided clean dataset.
Identified the 6 interrelated tables: Employee, JobDepartment, SalaryBonus, Qualification, Leaves, and Payroll.

### Database Design
Created a structured relational database.
Defined primary keys and foreign keys.
Established relationships between tables.

### Data Import
Imported the dataset into MySQL Workbench.
Verified data consistency and structure.

### Establishing Relationships
Linked tables using foreign key constraints.
Applied cascading rules to maintain data integrity.

### Data Analysis
Analyzed employee distribution across departments.
Evaluated salary ranges and total payroll expenditure.
Examined leave patterns and their impact on payroll.
Identified highest-paid roles and employees.
Reviewed qualification distribution.

### Insight Generation
Generated meaningful business insights.
Interpreted results to support HR and compensation decisions.

### Documentation & Presentation
Prepared ER diagram and relationship explanation.
Structured findings into a presentation format.
